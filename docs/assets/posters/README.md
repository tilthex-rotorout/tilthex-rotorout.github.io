Place video poster images here using the same folder and base filename as the matching video.

Examples:
- `assets/videos/main/main_part_1.mp4` -> `assets/posters/main/main_part_1.jpg`
- `assets/videos/trajectory/trajectory_01.mp4` -> `assets/posters/trajectory/trajectory_01.jpg`
- `assets/videos/outdoor/outdoor_01.mp4` -> `assets/posters/outdoor/outdoor_01.jpg`
- `assets/videos/hover/0deg/BTO-CL/1.mp4` -> `assets/posters/hover/0deg/BTO-CL/1.jpg`
- `assets/videos/hover/45deg/UTO-AL/36.mp4` -> `assets/posters/hover/45deg/UTO-AL/36.jpg`

Supported poster extensions:
- `.jpg`
- `.jpeg`
- `.png`
- `.webp`

If a matching poster is not found, the page falls back to `assets/posters/default-video-poster.svg`.
